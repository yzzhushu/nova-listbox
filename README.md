### Installation

Install the package in a Laravel Nova project via Composer:

```bash
composer require jshxl/list-box
```

### Requirements

- `php: ^7.3|^8.0`
- `laravel/framework: ^10.10`
- `laravel/nova: ^4.0`
- `jshxl/primevue: ^1.0.0`
